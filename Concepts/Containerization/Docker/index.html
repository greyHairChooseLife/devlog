권한

하나도 신경 안쓰고 있었다.

불편한 거라면 volume으로 설정한 디렉토리의 파일을 직접 손대려고 하면 권한 문제가 발생하는점이다.

로깅

하나도 신경 안쓰고 있었다. 뭐가 잘 안되면 docker logs 명령으로 살펴보기나 했지.

배포 환경에도 도커를 활용한다면 이렇게 무책임해서는 안되겠다.
